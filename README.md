# Impact Labs Website

## Setup

1.  Install [bundler](https://bundler.io/)
2.  Run `bundle install`
3.  Run `jekyll serve`

# Layout
The stuff that matters:
```
.
├── _config.yml
├── _data
├── _includes               Page elements that persist across the site (navbar, footer, head)
├── _layouts                All pages are rendered through here
├── _sass                   main.scss (edit this and not /assetscss/styles.scss)
├── _site                   Auto generated by Jekyll
├── assets                  Static assets (images, CSS and JS files)
└── pages
│   └── ...
```
